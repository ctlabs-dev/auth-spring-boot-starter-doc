## Soporte

Si encuentras algún problema o tienes preguntas:

- **Reportes de Bugs**: Abre un issue en el [repositorio de GitHub](https://github.com/ctlabs-dev/auth-spring-boot-starter/issues)
- **Preguntas**: Revisa primero el [FAQ](/support/faq), luego abre una discusión en GitHub
- **Solicitudes de Funcionalidades**: Abre un issue con la etiqueta `enhancement`

::tip
Antes de abrir un issue, por favor busca en los issues existentes para evitar duplicados.
::

## Contribuir

¡Damos la bienvenida a contribuciones! Así es como puedes ayudar:

### Formas de Contribuir

- 🐛 **Reportar bugs** con pasos detallados de reproducción
- 💡 **Sugerir funcionalidades** que mejorarían el starter
- 📝 **Mejorar documentación** (corregir errores tipográficos, aclarar conceptos, agregar ejemplos)
- 🔧 **Enviar pull requests** con correcciones de bugs o nuevas funcionalidades
- ⭐ **Dale estrella al repositorio** si lo encuentras útil

### Guías de Contribución

1. Haz fork del repositorio
2. Crea una rama de funcionalidad (`git checkout -b feature/funcionalidad-increible`)
3. Confirma tus cambios (`git commit -m 'Agregar funcionalidad increíble'`)
4. Empuja a la rama (`git push origin feature/funcionalidad-increible`)
5. Abre un Pull Request

::note
Asegúrate de seguir el estilo de código existente e incluir pruebas para nuevas funcionalidades.
::

## Seguridad

### Reportar Vulnerabilidades de Seguridad

Si descubres una vulnerabilidad de seguridad, por favor **NO** abras un issue público.

En su lugar, envíanos un email a: **security@ctlabs.dev**

Tomamos la seguridad en serio y responderemos prontamente para abordar cualquier vulnerabilidad.

### Mejores Prácticas de Seguridad

Al usar este starter en producción:
- ✅ Usa siempre HTTPS en producción
- ✅ Almacena secretos en variables de entorno (nunca los confirmes)
- ✅ Habilita verificación por email/teléfono (no uses Provider.NONE)
- ✅ Configura políticas de contraseña fuertes
- ✅ Actualiza regularmente las dependencias
- ✅ Monitorea logs de autenticación para actividad sospechosa
