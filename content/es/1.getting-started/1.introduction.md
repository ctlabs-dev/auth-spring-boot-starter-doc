::tip{title="Listo para Producción"}
Auth Starter es una infraestructura completa para autenticación y gestión de usuarios. Enfócate en tu lógica de negocio mientras nosotros manejamos la seguridad.
::

## Características Principales

- **🛡️ Autenticación Basada en JWT**: Tokens de acceso y renovación seguros con revocación automática.
- **🌍 Bilingüe por Diseño**: Soporte nativo para notificaciones y validaciones en múltiples idiomas.
- **📲 Notificaciones Inteligentes**: Integración perfecta con SMTP, Twilio y Brevo para Email, SMS y WhatsApp.
- **🔑 Seguridad Granular (RBAC)**: Modelo completo de Roles y Permisos inyectado en el contexto de Spring Security.
- **🚀 Configuración Sin Migraciones**: Gestión automática de tablas usando migraciones Flyway.

## Requisitos Previos

::note{title="Requisitos del Sistema"}
Antes de comenzar, asegúrate de que tu entorno cumple con los siguientes requisitos:
::

- **Spring Boot 4**: Construido para el ecosistema Spring más reciente.
- **Java 21 o superior**: Aprovecha las características y rendimiento de Java moderno.
- **PostgreSQL**: Base de datos recomendada (soporta metadatos JSONB y extensiones UUID).
- **Maven o Gradle**: Herramientas de compilación estándar para gestión de dependencias.
