## Principal

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.base-url` | `/api/auth` | Ruta base para todos los endpoints de autenticación. |
| `ctlabs.auth.default-role` | `USER` | Rol predeterminado para nuevos usuarios. |
| `ctlabs.auth.public-paths` | `[]` | Lista de rutas públicas para excluir de seguridad. |

## JWT

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.jwt.secret-key` | `null` | **Requerido**. Clave de firma (mín 256 bits). |
| `ctlabs.auth.jwt.expiration` | `15m` | Vida útil del token de acceso. |

## Token de Renovación

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.refresh-token.expiration` | `30d` | Vida útil del token de renovación. |

## Base de Datos

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.db.migration-enabled` | `true` | Habilitar migraciones Flyway. |

## Notificaciones por Email

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.notifications.mail.provider` | `NONE` | Proveedor de email: `NONE`, `SMTP`, `BREVO`. |
| `ctlabs.auth.notifications.mail.frontend-url` | `http://localhost:3000` | URL del frontend para enlaces de verificación de email. |
| `ctlabs.auth.notifications.mail.smtp.from-email` | `null` | Email del remitente SMTP. |
| `ctlabs.auth.notifications.mail.smtp.from-name` | `null` | Nombre del remitente SMTP. |
| `ctlabs.auth.notifications.mail.brevo.api-key` | `null` | Clave API de Brevo. |
| `ctlabs.auth.notifications.mail.brevo.base-url` | `https://api.brevo.com/v3` | URL base de API de Brevo (avanzado). |
| `ctlabs.auth.notifications.mail.brevo.verification-template-id` | `null` | ID de plantilla de verificación de email de Brevo. |
| `ctlabs.auth.notifications.mail.brevo.password-reset-template-id` | `null` | ID de plantilla de restablecimiento de contraseña de Brevo. |

## Usuario Admin

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.admin.enabled` | `false` | Crear usuario admin al inicio. |
| `ctlabs.auth.admin.email` | `null` | Dirección de email del admin. |
| `ctlabs.auth.admin.password` | `null` | Contraseña del admin. |
| `ctlabs.auth.admin.first-name` | `Admin` | Nombre del admin. |
| `ctlabs.auth.admin.last-name` | `System` | Apellido del admin. |
| `ctlabs.auth.admin.role` | `ADMIN` | Nombre del rol admin. |
| `ctlabs.auth.admin.permissions` | `[]` | Lista de permisos para el rol admin. |

## Verificación

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.verification.email-link-expiration` | `24h` | Expiración del enlace de verificación de email. |
| `ctlabs.auth.verification.phone-code-expiration` | `10m` | Expiración del código de verificación por teléfono. |
| `ctlabs.auth.verification.unit-minutes` | `minutes` | Etiqueta para minutos (i18n). |
| `ctlabs.auth.verification.unit-hours` | `hours` | Etiqueta para horas (i18n). |

## Contraseña

| Propiedad | Predeterminado | Descripción |
| :--- | :--- | :--- |
| `ctlabs.auth.password.validation-regex` | `^(?=.*[0-9])(?=.*[a-z])(?=.*[A-Z])(?=\\S+$).{8,20}$` | Regex de validación de contraseña. |
| `ctlabs.auth.password.validation-message` | `Password must be 8-20 characters...` | Mensaje de error de validación. |
