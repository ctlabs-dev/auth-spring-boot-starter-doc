## Validación de Contraseñas

Configura requisitos de contraseña usando un regex personalizado y mensaje de validación.

| Propiedad | Descripción | Predeterminado |
| :--- | :--- | :--- |
| `ctlabs.auth.password.validation-regex` | Patrón regex para validación de contraseña. | `^(?=.*[0-9])(?=.*[a-z])(?=.*[A-Z])(?=\\S+$).{8,20}$` |
| `ctlabs.auth.password.validation-message` | Mensaje de error mostrado cuando la contraseña no coincide con el regex. | `Password must be 8-20 characters...` |

**Requisitos predeterminados:**
- 8-20 caracteres de longitud
- Al menos un dígito
- Al menos una letra minúscula
- Al menos una letra mayúscula
- Sin espacios en blanco

**Ejemplo - Validación personalizada:**

::code-group
```properties [application.properties]
# Requiere 12+ caracteres, 1 dígito, 1 carácter especial
ctlabs.auth.password.validation-regex=^(?=.*[0-9])(?=.*[!@#$%^&*])(?=\\S+$).{12,}$
ctlabs.auth.password.validation-message=La contraseña debe tener al menos 12 caracteres con un número y un carácter especial
```

```yaml [application.yml]
# Requiere 12+ caracteres, 1 dígito, 1 carácter especial
ctlabs:
  auth:
    password:
      validation-regex: "^(?=.*[0-9])(?=.*[!@#$%^&*])(?=\\S+$).{12,}$"
      validation-message: La contraseña debe tener al menos 12 caracteres con un número y un carácter especial
```
::

## Usuario Admin

Crea un usuario administrador inicial automáticamente en el primer inicio. Esto es útil para arrancar tu aplicación con una cuenta administrativa.

| Propiedad | Descripción | Predeterminado |
| :--- | :--- | :--- |
| `ctlabs.auth.admin.enabled` | Si se debe crear usuario admin al iniciar. | `false` |
| `ctlabs.auth.admin.email` | Dirección de email del admin. | `null` |
| `ctlabs.auth.admin.password` | Contraseña del admin. | `null` |
| `ctlabs.auth.admin.first-name` | Nombre del admin. | `Admin` |
| `ctlabs.auth.admin.last-name` | Apellido del admin. | `System` |
| `ctlabs.auth.admin.role` | Nombre del rol admin. | `ADMIN` |
| `ctlabs.auth.admin.permissions` | Lista de permisos para el rol admin. | `[]` (vacío) |

**Ejemplo:**

::code-group
```properties [application.properties]
ctlabs.auth.admin.enabled=true
ctlabs.auth.admin.email=admin@miapp.com
ctlabs.auth.admin.password=AdminPassword123!
ctlabs.auth.admin.first-name=Admin
ctlabs.auth.admin.last-name=System
ctlabs.auth.admin.role=ADMIN
ctlabs.auth.admin.permissions[0]=users:write
ctlabs.auth.admin.permissions[1]=users:delete
ctlabs.auth.admin.permissions[2]=settings:manage
```

```yaml [application.yml]
ctlabs:
  auth:
    admin:
      enabled: true
      email: admin@miapp.com
      password: AdminPassword123!
      first-name: Admin
      last-name: System
      role: ADMIN
      permissions:
        - users:write
        - users:delete
        - settings:manage
```
::

::warning
**Mejores Prácticas de Seguridad:**
- Usa variables de entorno para valores sensibles
- Nunca confirmes contraseñas en el control de versiones
::

## Ejemplo Completo

::code-group
```properties [application.properties]
# Validación de Contraseñas
ctlabs.auth.password.validation-regex=^(?=.*[0-9])(?=.*[a-z])(?=.*[A-Z])(?=\\S+$).{8,20}$
ctlabs.auth.password.validation-message=La contraseña debe tener 8-20 caracteres, contener al menos un dígito, una minúscula, una mayúscula y sin espacios en blanco

# Usuario Admin (deshabilitar después de la primera ejecución)
ctlabs.auth.admin.enabled=true
ctlabs.auth.admin.email=admin@miapp.com
ctlabs.auth.admin.password=AdminPassword123!
ctlabs.auth.admin.first-name=Admin
ctlabs.auth.admin.last-name=System
ctlabs.auth.admin.role=ADMIN
ctlabs.auth.admin.permissions[0]=users:write
ctlabs.auth.admin.permissions[1]=users:delete
```

```yaml [application.yml]
ctlabs:
  auth:
    password:
      validation-regex: "^(?=.*[0-9])(?=.*[a-z])(?=.*[A-Z])(?=\\S+$).{8,20}$"
      validation-message: La contraseña debe tener 8-20 caracteres, contener al menos un dígito, una minúscula, una mayúscula y sin espacios en blanco

    admin:
      enabled: true
      email: admin@miapp.com
      password: AdminPassword123!
      first-name: Admin
      last-name: System
      role: ADMIN
      permissions:
        - users:write
        - users:delete
```
::
