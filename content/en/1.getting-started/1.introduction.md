::tip{title="Production Ready"}
Auth Starter is a complete infrastructure for authentication and user management. Focus on your business logic while we handle the security.
::

## Core Features

- **🛡️ JWT-Based Authentication**: Secure access and refresh tokens with automatic revocation.
- **🌍 Bilingual by Design**: Native support for notifications and validations in multiple languages.
- **📲 Smart Notifications**: Seamless integration with SMTP, Twilio, and Brevo for Email, SMS, and WhatsApp.
- **🔑 Granular Security (RBAC)**: Complete Role and Permission model injected into the Spring Security context.
- **🚀 Zero-Migration Setup**: Automatic table management using Flyway migrations.

## Prerequisites

::note{title="System Requirements"}
Before you begin, ensure your environment meets the following requirements:
::

- **Spring Boot 4**: Built for the latest Spring ecosystem.
- **Java 21 or higher**: Leverages modern Java features and performance.
- **PostgreSQL**: Recommended database (supports JSONB metadata and UUID extensions).
- **Maven or Gradle**: Standard build tools for dependency management.
