## Core

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.base-url` | `/api/auth` | Base path for all auth endpoints. |
| `ctlabs.auth.default-role` | `USER` | Default role for new users. |
| `ctlabs.auth.public-paths` | `[]` | List of public paths to exclude from security. |

## JWT

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.jwt.secret-key` | `null` | **Required**. Signing key (min 256 bits). |
| `ctlabs.auth.jwt.expiration` | `15m` | Access token lifespan. |

## Refresh Token

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.refresh-token.expiration` | `30d` | Refresh token lifespan. |

## Database

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.db.migration-enabled` | `true` | Enable Flyway migrations. |

## Email Notifications

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.notifications.mail.provider` | `NONE` | Email provider: `NONE`, `SMTP`, `BREVO`. |
| `ctlabs.auth.notifications.mail.frontend-url` | `http://localhost:3000` | Frontend URL for email verification links. |
| `ctlabs.auth.notifications.mail.smtp.from-email` | `null` | SMTP sender email. |
| `ctlabs.auth.notifications.mail.smtp.from-name` | `null` | SMTP sender name. |
| `ctlabs.auth.notifications.mail.brevo.api-key` | `null` | Brevo API key. |
| `ctlabs.auth.notifications.mail.brevo.base-url` | `https://api.brevo.com/v3` | Brevo API base URL (advanced). |
| `ctlabs.auth.notifications.mail.brevo.verification-template-id` | `null` | Brevo email verification template ID. |
| `ctlabs.auth.notifications.mail.brevo.password-reset-template-id` | `null` | Brevo password reset template ID. |

## Phone Notifications

| `ctlabs.auth.notifications.phone.provider` | `NONE` | Phone provider: `NONE`, `TWILIO`, `BREVO`. |
| `ctlabs.auth.notifications.phone.channel` | `SMS` | Channel: `SMS`, `WHATSAPP`. |
| `ctlabs.auth.notifications.phone.from-phone-number` | `null` | Phone number to send from. |

## Admin User

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.admin.enabled` | `false` | Create admin user at startup. |
| `ctlabs.auth.admin.email` | `null` | Admin email address. |
| `ctlabs.auth.admin.password` | `null` | Admin password. |
| `ctlabs.auth.admin.first-name` | `Admin` | Admin first name. |
| `ctlabs.auth.admin.last-name` | `System` | Admin last name. |
| `ctlabs.auth.admin.role` | `ADMIN` | Admin role name. |
| `ctlabs.auth.admin.permissions` | `[]` | List of permissions for admin role. |

## Verification

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.verification.email-link-expiration` | `24h` | Email verification link expiration. |
| `ctlabs.auth.verification.phone-code-expiration` | `10m` | Phone verification code expiration. |
| `ctlabs.auth.verification.unit-minutes` | `minutes` | Label for minutes (i18n). |
| `ctlabs.auth.verification.unit-hours` | `hours` | Label for hours (i18n). |

## Password

| Property | Default | Description |
| :--- | :--- | :--- |
| `ctlabs.auth.password.validation-regex` | `^(?=.*[0-9])(?=.*[a-z])(?=.*[A-Z])(?=\\S+$).{8,20}$` | Password validation regex. |
| `ctlabs.auth.password.validation-message` | `Password must be 8-20 characters...` | Validation error message. |
